# 🚀 Website Portofolio & Admin Panel

Aplikasi Portofolio dinamis berbasis web yang dilengkapi dengan Admin Panel untuk mengelola data Profile, Skills, Experience, Projects, dan Pesan Kontak secara langsung.

## 🛠️ Tech Stack
- **Backend:** Python, Flask Blueprint
- **Database:** TiDB Cloud (MySQL Serverless)
- **Email Service:** Resend API
- **Frontend:** HTML5, CSS3 (Custom Theme)
